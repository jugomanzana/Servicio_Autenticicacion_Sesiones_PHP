# Servicio_Autenticicacion_Sesiones_PHP
La aplicación implementa las siguientes funcionalidades requeridas:

1.
Pantalla de Login 
Formulario para que el usuario ingrese su nombre de usuario y contraseña.

Validación de las credenciales en el servidor utilizando un array predefinido en PHP (sin base de datos).


2.
Pantalla de Bienvenida 
Muestra un mensaje personalizado que incluye:

El nombre del usuario autenticado.

La hora actual.

Un mensaje de bienvenida específico.

3.
Pantalla de "No Tienes Permisos" 
Si un usuario no autenticado intenta acceder a la pantalla de bienvenida, será redirigido automáticamente a esta página.

4.
Funcionalidad para Cerrar Sesión 
Un enlace en la pantalla de bienvenida permite cerrar la sesión y redirigir al usuario al formulario de login.
